# Likalani Shadrack - Professional Graphic Design Portfolio

A modern, responsive portfolio website with integrated e-commerce functionality for graphic design services.

![Portfolio Preview](./assets/preview.png)

## ✨ Features

- 🎨 **Multi-Page Portfolio** - Home, About, Creations, Contact
- ☁️ **Cloudinary Integration** - Permanent image hosting with CDN
- 🔐 **User Authentication** - Netlify Identity for secure login
- 🛒 **Order System** - Clients can place orders for services
- 💳 **Payment Integration** - Stripe & PayPal for deposits
- 📊 **Admin Dashboard** - Manage orders and track projects
- ⏱️ **Timeline Tracking** - Clients can track project progress
- 📱 **Fully Responsive** - Works on all devices
- 🌐 **Free Hosting** - Deployed on Netlify
